Viper AI ships a set of production-ready components for building AI-native engineering interfaces. All components are copy-paste, fully typed, and built on Radix UI + Tailwind CSS.

---

## Agent Components

High-level components that wrap Viper's agent fleet with real-time streaming UI.

### CodeAgent

Displays live code generation output with syntax highlighting and diff view.

```tsx
import { CodeAgent } from '@/components/agents/code-agent'

<CodeAgent
  prompt="Add Stripe webhook handler"
  onComplete={(output) => console.log(output)}
/>
```

### ReviewAgent

Shows PR review output with inline comment threading.

### PMAgent

Renders ticket creation forms with AI-generated acceptance criteria.

### OpsAgent

Pipeline health monitor with real-time log streaming.

## UI Components

### Button

```tsx
import { Button } from '@/components/ui/button'

<Button variant="default">Get Started</Button>
<Button variant="outline">Learn More</Button>
<Button variant="ghost">Cancel</Button>
```

### ShimmerButton

An animated shimmer button for CTAs.

```tsx
import ShimmerButton from '@/components/ui/shimmer-button'

<ShimmerButton>Request Early Access</ShimmerButton>
```

### Card

Dark-themed card with hover border animation.

```tsx
<div className="bg-[#0a0a0a] border border-white/10 rounded-lg p-8 hover:border-white/25 transition-all duration-300">
  <h3>Card Title</h3>
  <p>Card content goes here.</p>
</div>
```
