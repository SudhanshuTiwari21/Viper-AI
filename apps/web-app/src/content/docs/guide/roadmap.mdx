The Viper AI roadmap is public. Here's what we're working on and what's coming next.

---

## In Progress

### v0.9.0 — Multi-repo support

- Cross-repository context sharing between microservices
- Monorepo-aware agent coordination
- Unified PR view across repositories

### v0.9.0 — Agent marketplace

- Community-contributed agent templates
- One-click agent configuration presets
- Share and fork agent workflows

## Planned

### Q3 2025

- **Web dashboard** — Visual interface for agent monitoring and control
- **Slack bot** — Run Viper commands directly from Slack
- **CI/CD deep integration** — Native GitHub Actions and CircleCI plugins

### Q4 2025

- **Self-hosted option** — Run Viper entirely on your own infrastructure
- **Custom agent SDK** — Build and deploy your own specialized agents
- **Audit dashboard** — Full visual history of every agent decision

## Community Requests

These features have been requested by users and are under consideration:

- GitLab self-hosted support
- Azure DevOps integration
- Bitbucket support
- Agent-to-agent delegation protocols

Have a feature request? [Open an issue on GitHub](#) or vote on existing requests.
