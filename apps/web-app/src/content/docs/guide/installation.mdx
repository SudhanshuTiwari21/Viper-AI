Get started with Viper AI by installing the CLI and connecting your workspace to the agent fleet.

---

## Requirements

Before you begin, make sure you have:

- **Node.js** 18.0 or later
- **npm** 9.0 or later
- A GitHub or GitLab account for code integration

## Install the CLI

Install the Viper CLI globally via npm:

```bash
npm install @viper-ai/cli -g
```

## Initialize your workspace

Run `viper init` inside your project directory:

```bash
cd my-project
viper init
```

This creates a `viper.config.ts` file at the root of your project with sensible defaults.

## Connect your workspace

Authenticate and connect Viper to your repository:

```bash
viper connect --workspace my-project
```

You'll be prompted to authorize Viper with your GitHub or GitLab account.

## Verify installation

Confirm everything is working:

```bash
viper status
```

You should see output like:

```
✓ CLI version: 0.8.2
✓ Workspace: my-project
✓ Agents: 4 active
✓ Context: indexed (247 files)
```
