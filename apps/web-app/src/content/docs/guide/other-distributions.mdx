Viper AI is part of a growing ecosystem of open, AI-native engineering tools. Here are some other distributions worth knowing about.

---

## shadcn/ui

The original open component distribution. Viper's component architecture is directly inspired by shadcn/ui's approach — copy-paste components you own forever.

[shadcn.com](https://ui.shadcn.com)

## Animate UI

A distribution of animated React components built with Tailwind CSS and Framer Motion. Many of Viper's motion primitives are ported or inspired by Animate UI.

[animate-ui.com](https://animate-ui.com)

## Magic UI

A collection of re-usable components built with React, TypeScript, Tailwind CSS, and Framer Motion. Focused on beautiful, production-ready UI.

## Radix UI

The accessible primitive layer that powers most of Viper's interactive components. All keyboard navigation and ARIA attributes come from Radix.

[radix-ui.com](https://www.radix-ui.com)

## Framer Motion

The animation library used throughout Viper for transitions, gestures, and scroll-driven effects.

[framer.com/motion](https://www.framer.com/motion)
