All notable changes to Viper AI are documented here. Viper follows [Semantic Versioning](https://semver.org/).

---

## v0.8.2 — Latest

**Released:** April 2025

- Added MCP server support for Cursor and VS Code
- Improved context indexing speed by 3x for large repositories
- Fixed agent rollback not restoring file permissions
- Added `viper logs` command for real-time agent output streaming

## v0.8.0

**Released:** March 2025

- Introduced the **Review Agent** with full PR context analysis
- Added Jira and Linear ticket sync for the PM Agent
- Improved multi-file context window for the Code Agent
- New `viper.config.ts` schema with full TypeScript support

## v0.7.5

**Released:** February 2025

- Fixed OAuth token refresh on long-running sessions
- Added `--dry-run` flag to all agent commands
- Improved error messages for missing workspace config
- Performance improvements to the shared context graph

## v0.7.0

**Released:** January 2025

- Initial public early access release
- Code Agent, PM Agent, and Ops Agent available
- GitHub integration for PRs and code review
- Basic CLI with `init`, `run`, and `status` commands
